ComfyUI Meta Viewer - Help & Shortcuts

About CMV

CMV is a metadata extraction tool specifically designed for images generated with ComfyUI and Stable Diffusion WebUI. It extracts embedded positive/negative prompts, sampler parameters, models, and visualizes the complete node generation workflow graph.

Features

  • 📂 Source Monitor: Watch local and desktop-synced folders and reconcile missed changes.
  • 💾 Originals Storage: Added files are stored in the database and indexed when opened.
  • 📊 Interactive Graph: Navigate and inspect active workflow generation nodes.
  • 🔍 EXIF extraction: Extracts raw metadata chunks from PNG, JPEG, WEBP files.

How to use

Drag & drop images or videos anywhere, or click Open Folder to connect a directory. Source controls toggle monitoring, subfolder scanning, and manual reconciliation. Reset Index rebuilds disposable data; Factory Reset also forgets saved sources and preferences.
