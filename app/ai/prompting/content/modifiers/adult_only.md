# Modifier: adult only

Apply only when the user has explicitly selected an adult-only workflow.

## Hard boundaries

- Every depicted person must be unambiguously adult.
- Never combine adult-oriented framing with childlike, school-age, youthful, or ambiguous-age cues.
- Do not infer an adult-only request from vague wording, clothing, or mood.
- Keep descriptions precise and non-vulgar.
- Respect the selected provider, model, application, and workflow restrictions.
- Do not add policy commentary to the final prompt.
