Keyboard navigation

Keep your hands on the keyboard while reviewing.

←↑↓→NavigateMove the active asset through the grid
SpaceSelectToggle selection for the active asset
FFavoriteAdd or remove the active asset from favorites
1–5RateSet the active asset rating instantly
DeleteDelete filesMove selected physical files from the computer to the Recycle Bin / Trash
EscClear or returnClear a selection in place; with nothing selected, return to the start
02

Selection

Use Select Mode for bulk actions.

Double-click

Double-click an asset to enter Select Mode and select it.

Range selection

Hold Shift while selecting to include the assets between two points.

Bulk actions

Favorite, rate, edit, remove from the index, or delete selected physical files from the computer.
